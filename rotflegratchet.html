Autobot Ratchet (Legend) [RotF]

Allegiance : Autobot
Size : Legend
Difficulty of Transformation : Very Easy
Color Scheme : Pale tan, black, and some silver and metallic rusty brownish red
Rating : 5.0

(NOTE: Because this is a repaint, this is not a full-blown review. This mainly covers any changes made to the mold and the color scheme, and merely compares it to the original Movie Legends class Ratchet toy. For a review on the mold itself, read the review of the original Movie Legends class Ratchet toy here .)

For Revenge of the Fallen, Ratchet gets a bit more of a "desert-themed" of a color scheme, given where a large part of that movie takes place. It works quite well for a Hummer, and is pretty fitting. The tan goes quite well with the black, but I especially love the metallic rusty brownish red that's used for the grill, top rack, and the robot chest-- it's a really eye-catching color, and it goes great with the main tan color. I'm not too fond of the silver used for a few of his accents like the "heartbeat" stripe, though-- it barely can be seen against the tan at all in average light, and just doesn't contrast well enough with the other colors. A bit more of the rusty red would've worked better, methinks. One other small complaint about the deco job is it's hardly appropriate for an emergency vehicle, but that's already been done on the toy, so something different this time works for me.
No mold changes have been made to RotF Legend class Ratchet.
RotF Legend class Ratchet is a nice "desert" take on Hummer's color scheme, and has some really nice colors offsetting it (except for the silver, which is a bit hard to see against the tan). Not quite as good as the Rescue Ratchet deco, but almost as good, methinks, and definitely better than his movie-accurate deco.
