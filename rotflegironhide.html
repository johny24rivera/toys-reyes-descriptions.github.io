Ironhide (Enforcer; Legend)

Allegiance : Autobot
Size : Legend
Difficulty of Transformation : Very Easy
Color Scheme : Moderately light red, charcoal gray, and some metallic bronze, light sky blue, and silver
Rating : 6.7

(NOTE: Because this is a repaint, this is not a full-blown review. This mainly covers any changes made to the mold and the color scheme, and merely compares it to the original Movie Legends class Ironhide. For a review on the mold itself, read the review of the original Movie Legends class Ironhide here .)

It's been a while, but Movie Ironhide finally gets done up in a G1-esque color scheme, with light red being the dominant color. The red is a bit light for something like a Topkick truck though, honestly. Charcoal gray makes up the main secondary color, and it contrasts fairly well with the light red, but honestly a straight black would've looked better and created better contrast, in my opinion. The nice "rectangular grey hood" paint app with the Autobot symbol on one side looks pretty nice, though and really helps to break up all red in vehicle mode. The metallic bronze is my personal favorite color of the bunch, even if it's only used for the windows-- it's a really nice shade, and contrasts excellently with both the red and the charcoal gray. To a slightly lesser extent (since it's used on TFs so often), I have the same thoughts about the silver used on the chest in robot mode. He could also use a few more paint apps, particularly on his arm-guns. Red arm-guns just don't look too great.
No mold changes have been made to Enforcer Ironhide.
Enforcer Ironhide is a nice bit of a nod to the character's G1 coloration, but honestly I don't think the red looks all that great on this mold, in this shade. The charcoal gray, as opposed to a straight black, doesn't help much, either. It's not bad by any stretch, but I'd recommend Ironhide's movie-accurate color scheme over this one.
