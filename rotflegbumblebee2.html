Bumblebee (Recon)

Allegiance : Autobot
Size : Legend
Difficulty of Transformation : Very Easy
Color Scheme : Sparkly dull yellow, black, and some metallic silvery blue and silver
Rating : 7.5

(NOTE: Because this is a repaint, this is not a full-blown review. This mainly covers any changes made to the mold and the color scheme, and merely compares it to the original RotF Legends class Bumblebee. For a review on the mold itself, read the review of the original RotF Legends class Bumblebee here .)

Recon Bumblebee isn't really a repaint so much as a bit of a variant. The most visible difference between this and the original RotF Legends Bumblebee is that this version has a pretty neat NEST-ish Autobot symbol deco job done up in a triangle on the hood, with Bumblebee's two stripes on the hood being appropriately curved so that they don't intersect with that symbol. There's also graffiti-esque symbols on the car doors. There's a few other, less obvious changes-- the windows are now a lighter shade of silvery blue (which in my opinion doesn't look quite as good as the original's window paint shade). The headlights on the front aren't painted on this version, and in robot mode Bumblebee's Autobot symbol is on the opposite leg. Whee.
No mold changes have been made to Recon Bumblebee.
I think the nifty deco changes to Bumblebee's stripes and sides are kinda cool, but honestly he's otherwise waaaay too similar to the original RotF Legends release to justify purchasing again for all but the most hardcore completists. If you don't have the original, though, I'd recommend this slightly over it (though in turn I'd recommend the second mainline repaint of this mold, Stealth Bumblebee, more than this one).
